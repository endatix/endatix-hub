Endatix Form Embed - Basic Example

How to Use

This example demonstrates embedding an Endatix form using a static iframe. Simply replace the form ID in the src attribute with your actual form ID.

Embed URL Pattern: http://localhost:3000/embed/{formId}

With Prefilling: http://localhost:3000/embed/{formId}?token={your-token}
