# What is Endatix Hub?

Endatix Hub is a comprehensive, web-based form management platform that empowers organizations to create, customize, and manage forms without any coding knowledge. Built on top of SurveyJS - the most customizable form builder library - Endatix Hub provides everything you need to build professional form solutions.

## Platform Overview

Endatix Hub is designed to be the complete solution for form-centric applications, offering:

- **Advanced Form Builder**: Create complex forms with 20+ question types
- **Custom Question Types**: Build reusable components with drag-and-drop editor
- **Form Management**: Centralized backend for organizing forms and submissions
- **White-labeling**: Full control over appearance and branding
- **Integrations**: Connect with CRM, email marketing, and custom systems
- **Data Privacy**: HIPAA, GDPR, and CCPA compliant

## Key Differentiators

Unlike other form management systems, Endatix Hub:

- **No Coding Required**: Business users can create complex forms without developers
- **Built on SurveyJS**: Leverages the most powerful form builder library
- **Enterprise Ready**: Designed for large organizations with complex needs
- **Customizable**: White-label the entire platform to match your brand
- **Scalable**: Handle thousands of forms and submissions
- **Secure**: Enterprise-grade security and compliance features

## Core Concepts

### Forms and Form Definitions
In Endatix Hub, a **form** is the main entity that represents a data collection form. Each form has:
- **Form Definition**: The structure, questions, and logic (powered by SurveyJS JSON)
- **Form Settings**: Configuration for behavior, appearance, and integrations
- **Form Status**: Draft, published, or archived states

### Submissions
A **submission** is a completed response to a form, containing:
- **Response Data**: The actual answers provided by users
- **Metadata**: Timestamp, user agent, IP address, etc.
- **Status**: Processing, completed, or error states

### Custom Question Types
Endatix Hub allows you to create custom question types:
- **Preconfigured Questions**: Save time with reusable templates
- **Composite Questions**: Build complex input blocks (addresses, cascading dropdowns)
- **Advanced Features**: Add custom UI or device inputs (barcodes, NFC, GPS)

## Architecture Overview

Endatix Hub follows a modern web architecture:

- **Frontend**: Next.js-based web application with React components
- **Backend**: RESTful API for form and submission management
- **Database**: Secure storage for forms, submissions, and user data
- **Integrations**: Webhook system for external system connections
- **Security**: JWT authentication, role-based access control, encryption
